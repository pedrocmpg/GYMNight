# Mapeamento de Cardios e Nível de Esforço (PSE 1 a 10) - GYMNight

| Tipo de Cardio | Intensidade | PSE Média | Descrição do Esforço Prático |
| :--- | :---: | :---: | :--- |
| **Máquinas (Academia)** | | | |
| Esteira (Caminhada Plana) | Baixa | 2 - 3 | Respiração normal, dá para cantar uma música. |
| Bicicleta Ergométrica (Leve) | Baixa | 2 - 3 | Esforço mínimo, usado para soltar a musculatura. |
| Elíptico (Ritmo constante) | Moderada | 4 - 5 | Respiração um pouco mais funda, mas consegue conversar. |
| Esteira (Caminhada Rápida) | Moderada | 4 - 6 | Começa a suar, a conversa já fica entrecortada. |
| Esteira (Caminhada Inclinada)| Moderada/Alta | 6 - 7 | Frequência cardíaca sobe bem, pernas queimam mais. |
| Máquina de Remo (Ritmo Médio)| Moderada/Alta | 6 - 7 | Exige corpo inteiro, fôlego fica pesado rapidamente. |
| Corrida Contínua (Trote) | Alta | 6 - 8 | Foco na respiração, impossível manter uma conversa longa. |
| Simulador de Escada | Alta | 7 - 8 | Esforço muscular e cardiovascular intensos contínuos. |
| Air Bike (Tiros) | Máxima | 9 - 10| Sensação de morte iminente. Respiração ofegante, dor muscular. |
| Esteira (HIIT/Tiros) | Máxima | 9 - 10| Esforço total por curtos períodos. Não dá para falar nada. |

| **Peso Corporal / Outdoor** | | | |
| :--- | :---: | :---: | :--- |
| Caminhada ao ar livre | Baixa | 2 - 3 | Atividade de base. |
| Ciclismo (Passeio) | Baixa/Mod | 3 - 5 | Depende do terreno, mas no plano é bem tranquilo. |
| Pular Corda (Constante) | Alta | 7 - 8 | Coordenação e impacto, eleva o BPM muito rápido. |
| Burpees / Polichinelos | Alta | 8 - 9 | Exige muito oxigênio, esgota rápido em séries longas. |
| Sprints na Rua | Máxima | 9 - 10| Velocidade máxima, exaustão do sistema nervoso central. |

| **Esportes (Dinâmicos)** | | | |
| :--- | :---: | :---: | :--- |
| Vôlei | Variável | 4 - 7 | Esforço intermitente. Explosão e pausas. |
| Jiu-Jitsu (Rola) | Alta | 7 - 9 | Tensão isométrica absurda misturada com cardio. |
| Futebol | Alta | 7 - 9 | Tiros curtos constantes misturados com trote ativo. |
| Basquete | Alta | 8 - 9 | Transições muito rápidas de quadra, picos de BPM. |
| Boxe / Muay Thai (Saco/Sparring)| Máxima | 8 - 10| Exige ombros, core e fôlego de forma brutal. |
